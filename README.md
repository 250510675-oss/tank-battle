# 坦克大战 (Tank Battle)

一个用HTML5 Canvas编写的趣味坦克对战游戏！

## 🎮 游戏玩法

- **控制你的坦克**：使用 WASD 键或方向键移动
- **射击**：按空格键发射炮弹
- **目标**：消灭所有敌方坦克并保护你的基地
- **生存**：被击中会失去生命，基地被摧毁则游戏结束

## 📋 操作说明

### 键盘控制
- `W` / `↑` - 向上移动
- `A` / `←` - 向左移动
- `S` / `↓` - 向下移动
- `D` / `→` - 向右移动
- `空格键` - 射击
- `P` - 暂停/继续游戏

### 移动设备支持
- 滑动屏幕控制坦克移动
- 轻触屏幕进行射击

## 🚀 如何游玩

1. 点击"开始游戏"按钮
2. 使用键盘或触摸控制你的坦克
3. 消灭所有敌方坦克进入下一关
4. 小心不要被敌方坦克击中或撞到！
